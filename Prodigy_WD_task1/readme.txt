#Interactive Landing Page
Overview

This project is a simple responsive landing page built using HTML, CSS, and JavaScript.
It includes a fixed navigation bar with interactive effects and a structured footer.

#Features

Fixed navigation bar visible on all pages

Navbar changes style when scrolling

Hover effects on navigation links

Smooth section navigation

Responsive layout

Sticky footer positioned at the bottom

Clean and structured code

Technologies Used

HTML5 – Page structure

CSS3 – Styling and layout

JavaScript – Scroll-based interactivity
