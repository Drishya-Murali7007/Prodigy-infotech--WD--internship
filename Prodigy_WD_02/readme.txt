##Stopwatch Web Application

A responsive and fully functional Stopwatch Web App built using HTML, CSS, and JavaScript.

This project demonstrates real-time time tracking, DOM manipulation, and interactive UI design.

## Features

Start, Stop, and Reset functionality

Millisecond precision

Lap recording with structured display

Responsive layout

Interactive hover and click effects

Decorative layered UI design
